## 소개

**Custo**는 일본 아이돌 그룹 [CUTIE STREET](https://cutiestreet.asobisystem.com)(큐티스트리트)를 덕질하면서 일본어를 배울 수 있는 학습 사이트입니다.

- 🎤 **멤버 이름**으로 히라가나·카타카나 연습
- 🎵 **가사**로 단어와 문법 학습
- 📱 SNS, 현장, 팬레터 등 **덕질 실전 일본어**
- 💡 **퀴즈 & 플래시카드**로 인터랙티브 학습

## 콘텐츠

| 카테고리 | 내용 |
|----------|------|
| **초급** | 히라가나, 카타카나, 기초 표현 |
| **덕질 일본어** | 현장 용어, SNS 일본어, 팬레터 쓰기 |
| **가사로 배우기** | かわいいだけじゃだめですか？, Hello Hello Mirai, Happy Sekai! |
| **멤버** | 8인 프로필 + 한자 읽기 연습 |
| **디스코그래피** | 싱글 목록 + 곡 소개 |
| **스케줄** | 활동 일정 |

## 기술 스택

- [VitePress](https://vitepress.dev/) — 정적 사이트 생성
- [Vue 3](https://vuejs.org/) — 인터랙티브 컴포넌트 (Quiz, Flashcard, MemberCard)
- [GitHub Pages](https://pages.github.com/) — 호스팅

## 로컬 개발

```bash
npm install
npm run docs:dev     # 개발 서버 (localhost:5173)
npm run docs:build   # 프로덕션 빌드
npm run docs:preview # 빌드 프리뷰
```
