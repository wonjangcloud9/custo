일본어 입력 가이드 ​

스마트폰이나 PC에서 일본어를 입력하는 방법을 알아봅시다! SNS에서 일본어로 댓글 달 때 필수!

스마트폰 설정 ​

iPhone ​

  1. 설정 → 일반 → 키보드 → 키보드 → 새로운 키보드 추가
  2. 일본어 선택 → 로마자 선택
  3. 키보드 전환: 지구본 아이콘 탭

Android ​

  1. 설정 → 일반 관리 → 언어 및 입력 → 화면 키보드
  2. 사용 중인 키보드(Gboard 등) → 언어 → 일본어 추가
  3. 키보드 전환: 스페이스바 길게 누르기

PC 설정 ​

Windows ​

  1. 설정 → 시간 및 언어 → 언어 및 지역 → 언어 추가
  2. 일본어 추가
  3. 전환: Win + Space 또는 Alt + Shift

Mac ​

  1. 시스템 설정 → 키보드 → 입력 소스 → 편집
  2. 일본어 - 로마자 추가
  3. 전환: Ctrl + Space 또는 메뉴바 입력 소스 클릭

로마자 입력법 ​

일본어 키보드에서 로마자를 치면 자동으로 히라가나로 변환됩니다!

주의할 입력 ​

입력결과설명

카타카나 변환 ​

히라가나를 입력한 후 스페이스바를 누르면 변환 후보가 나와요. 카타카나를 선택하면 됩니다!

입력스페이스 →선택
raib uらいぶ →ライブ
chekiちぇき →チェキ
aidoruあいどる →アイドル
