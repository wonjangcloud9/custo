# 형용사 (形容詞)

일본어 형용사는 2종류! 큐티스트리트를 칭찬할 때 꼭 필요한 표현들이에요.

## い형용사

「い」로 끝나는 형용사입니다.

| 일본어 | 읽기 | 뜻 | 예문 |
|--------|------|-----|------|
| かわいい | 카와이이 | 귀여운 | 遥花ちゃんはかわいい！ |
| かっこいい | 캇코이이 | 멋진 | ダンスがかっこいい！ |
| 楽しい | 타노시이 | 즐거운 | ライブが楽しい！ |
| すごい | 스고이 | 대단한 | 歌がすごい！ |
| 新しい | 아타라시이 | 새로운 | 新しい曲が好き！ |
| 嬉しい | 우레시이 | 기쁜 | 会えて嬉しい！ |
| 近い | 치카이 | 가까운 | 席が近い！ |
| 高い | 타카이 | 높은/비싼 | グッズが高い！ |

### い형용사 활용

| 형태 | 만드는 법 | 예 (かわいい) | 뜻 |
|------|----------|-------------|-----|
| 현재 긍정 | そのまま | かわいい | 귀여워 |
| 현재 부정 | ~い → ~くない | かわいくない | 안 귀여워 |
| 과거 긍정 | ~い → ~かった | かわいかった | 귀여웠어 |
| 과거 부정 | ~い → ~くなかった | かわいくなかった | 안 귀여웠어 |
| 정중형 | + です | かわいいです | 귀여워요 |

### 예문

```
今日のライブは楽しかったです！
(쿄노 라이브와 타노시캇타데스!)
오늘 라이브는 즐거웠어요!

新しい曲がすごくかわいい！
(아타라시이 쿄쿠가 스고쿠 카와이이!)
새 곡이 엄청 귀여워!
```

> 💡 「すごく」처럼 い형용사를 부사로 쓸 때는 ~い → ~く로 바꿔요!

## な형용사

명사를 수식할 때 「な」를 붙이는 형용사입니다.

| 일본어 | 읽기 | 뜻 | 예문 |
|--------|------|-----|------|
| 素敵 | 스테키 | 멋진 | 素敵なステージ！ |
| 大好き | 다이스키 | 정말 좋아하는 | 大好きなグループ！ |
| 元気 | 겐키 | 활발한 | 元気なパフォーマンス！ |
| きれい | 키레이 | 예쁜/깨끗한 | きれいな衣装！ |
| 有名 | 유메이 | 유명한 | 有名なアイドル |
| 大切 | 타이세츠 | 소중한 | 大切な推し！ |
| 特別 | 토쿠베츠 | 특별한 | 特別なライブ！ |

### な형용사 활용

| 형태 | 만드는 법 | 예 (素敵) | 뜻 |
|------|----------|----------|-----|
| 현재 긍정 | + だ/です | 素敵です | 멋져요 |
| 현재 부정 | + じゃない | 素敵じゃない | 안 멋져 |
| 과거 긍정 | + だった | 素敵だった | 멋졌어 |
| 과거 부정 | + じゃなかった | 素敵じゃなかった | 안 멋졌어 |
| 명사 수식 | + な + 명사 | 素敵な笑顔 | 멋진 미소 |

### 예문

```
キューティーストリートは素敵なグループです。
(큐티스트리토와 스테키나 구룹푸데스.)
큐티스트리트는 멋진 그룹이에요.

大好きなメンバーに会えて嬉しい！
(다이스키나 멤바니 아에테 우레시이!)
정말 좋아하는 멤버를 만나서 기뻐!
```

## 덕질 필수 형용사 모음

멤버를 칭찬할 때 쓰면 좋은 표현들!

| 일본어 | 뜻 | 활용 예 |
|--------|-----|---------|
| かわいい | 귀여운 | 遥花ちゃんかわいい！ |
| かっこいい | 멋진 | ダンスかっこいい！ |
| 素敵 | 멋진 (격식) | 素敵なステージでした |
| 最高 | 최고 | ライブ最高！ |
| すごい | 대단한 | 歌すごい！ |
| 上手 | 잘하는 | 歌が上手！ |
| きれい | 예쁜 | 衣装がきれい！ |
| 優しい | 다정한 | 愛花ちゃんは優しい |
