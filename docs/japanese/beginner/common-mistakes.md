# 자주 틀리는 표현

한국인이 일본어를 배울 때 자주 혼동하는 표현들을 정리했어요!

## 1. は vs が

가장 많이 헷갈리는 조사!

| 상황 | 올바른 표현 | 설명 |
|------|------------|------|
| 자기소개 | 私**は**ファンです | 주제 제시 (나는~) |
| 누구냐고 물을 때 | 誰**が**推しですか？ | 새 정보 강조 (누가~?) |
| 좋아하는 것 | 遥花ちゃん**が**好きです | 감정의 대상은 が |
| 설명할 때 | 遥花ちゃん**は**かわいいです | 주제에 대해 설명 |

> 💡 핵심: **は** = 이미 알고 있는 주제, **が** = 새로운 정보/강조

## 2. に vs で

| 예문 | 조사 | 이유 |
|------|------|------|
| ライブ**に**行く | に | 이동의 목적지 |
| 渋谷**で**ライブを見る | で | 동작이 일어나는 장소 |
| 3時**に**始まる | に | 시간 |
| 電車**で**行く | で | 수단 |

> 💡 핵심: **に** = 도착점/시간, **で** = 동작 장소/수단

## 3. 발음 주의

| 문자 | 잘못된 발음 | 올바른 발음 | 설명 |
|------|-----------|-----------|------|
| は (조사) | ha | **wa** | 조사일 때만 wa |
| を | wo | **o** | 거의 o로 발음 |
| ん | n | **상황에 따라** | 뒤 글자에 따라 m/n/ng |
| し | si | **shi** | 영어 sh 발음 |
| つ | tu | **tsu** | ts 발음 |
| ふ | hu | **fu** | 입술 불며 f 비슷 |

## 4. い형 vs な형 혼동

겉모습은 비슷한데 종류가 다른 형용사!

| 단어 | 종류 | 주의점 |
|------|------|--------|
| かわいい | い형 | ○ 정상적인 い형 |
| きれい | **な형** | ✗ い로 끝나지만 な형! |
| 嫌い (키라이) | **な형** | ✗ い로 끝나지만 な형! |
| 大きい | い형 | ○ 정상적인 い형 |
| 有名 (유메이) | な형 | ○ 정상적인 な형 |

> 💡 「きれい」「嫌い」는 い로 끝나지만 な형용사! 틀리기 쉬우니 외우세요.

## 5. ます형 과거 혼동

| 형태 | 올바른 | 흔한 실수 |
|------|--------|----------|
| 긍정 과거 | 行き**ました** | 行き**ますた** ✗ |
| 부정 현재 | 行き**ません** | 行き**ますん** ✗ |
| 부정 과거 | 行き**ませんでした** | 行き**ましたない** ✗ |

## 6. 「大丈夫」만능 아님!

한국어 "괜찮아"처럼 만능이 아니에요.

| 상황 | 일본어 | 한국어 |
|------|--------|--------|
| 거절 | 大丈夫です | 괜찮습니다 (됐어요) |
| 걱정 | 大丈夫？ | 괜찮아? |
| 허락 | ✗ 大丈夫 | ○ いいですよ (돼요) |

## 7. 숫자 불규칙 읽기

| 숫자 | 흔한 실수 | 올바른 | 
|------|----------|--------|
| 4시 | しじ ✗ | **よじ** |
| 7시 | ななじ ✗ | **しちじ** |
| 9시 | きゅうじ ✗ | **くじ** |
| 1명 | いちにん ✗ | **ひとり** |
| 2명 | ににん ✗ | **ふたり** |
| 20일 | にじゅうにち ✗ | **はつか** |
