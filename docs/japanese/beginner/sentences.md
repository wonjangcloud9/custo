# 문장 만들기 (文を作ろう)

조사, 동사, 형용사를 조합해서 일본어 문장을 만들어봅시다! 큐티스트리트 덕질 문장으로 연습해요.

## 일본어 문장 구조

일본어는 **SOV** (주어-목적어-동사) 순서입니다.

```
한국어: 나는 라이브를 봐요
일본어: 私は ライブを 見ます
        (S)   (O)    (V)
```

> 💡 한국어와 어순이 같아서 배우기 쉬워요!

## 기본 패턴

### 패턴 1: A는 B입니다 (A는 B예요)

```
[명사]は [명사]です。
```

| 일본어 | 발음 | 뜻 |
|--------|------|-----|
| 私はファンです | 와타시와 팬데스 | 나는 팬이에요 |
| 遥花ちゃんはメンバーです | 하루카짱와 멤바데스 | 하루카는 멤버예요 |
| キューティーストリートはアイドルです | 큐티스트리토와 아이도루데스 | 큐티스트리트는 아이돌이에요 |

### 패턴 2: A는 B가 C (형용사)

```
[명사]は [명사]が [형용사]です。
```

| 일본어 | 발음 | 뜻 |
|--------|------|-----|
| 遥花ちゃんはダンスが上手です | 하루카짱와 단스가 조즈데스 | 하루카는 춤을 잘해요 |
| 里紗ちゃんは笑顔がかわいいです | 리사짱와 에가오가 카와이이데스 | 리사는 미소가 귀여워요 |
| ライブは雰囲気がすごいです | 라이브와 훈이키가 스고이데스 | 라이브는 분위기가 대단해요 |

### 패턴 3: A를 B합니다 (동사)

```
[명사]を [동사ます형]。
```

| 일본어 | 발음 | 뜻 |
|--------|------|-----|
| 曲を聞きます | 쿄쿠오 키키마스 | 곡을 들어요 |
| チェキを撮ります | 체키오 토리마스 | 체키를 찍어요 |
| ファンレターを書きます | 팬레타오 카키마스 | 팬레터를 써요 |

### 패턴 4: A에서 B를 C합니다 (장소+동사)

```
[장소]で [명사]を [동사ます형]。
```

| 일본어 | 발음 | 뜻 |
|--------|------|-----|
| 渋谷でライブを見ます | 시부야데 라이브오 미마스 | 시부야에서 라이브를 봐요 |
| 物販でグッズを買います | 붓판데 굿즈오 카이마스 | 물판에서 굿즈를 사요 |
| 特典会でチェキを撮ります | 토쿠텐카이데 체키오 토리마스 | 특전회에서 체키를 찍어요 |

### 패턴 5: ~하고 싶다 (~たい)

```
[명사]を [동사ます형 - ます + たい]。
```

| 일본어 | 발음 | 뜻 |
|--------|------|-----|
| ライブに行きたいです | 라이브니 이키타이데스 | 라이브에 가고 싶어요 |
| メンバーに会いたいです | 멤바니 아이타이데스 | 멤버를 만나고 싶어요 |
| 新曲を聞きたいです | 싱쿄쿠오 키키타이데스 | 신곡을 듣고 싶어요 |
