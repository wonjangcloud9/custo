# 일본어 입력 가이드

스마트폰이나 PC에서 일본어를 입력하는 방법을 알아봅시다! SNS에서 일본어로 댓글 달 때 필수!

## 스마트폰 설정

### iPhone

1. **설정** → **일반** → **키보드** → **키보드** → **새로운 키보드 추가**
2. **일본어** 선택 → **로마자** 선택
3. 키보드 전환: 지구본 아이콘 탭

### Android

1. **설정** → **일반 관리** → **언어 및 입력** → **화면 키보드**
2. 사용 중인 키보드(Gboard 등) → **언어** → **일본어** 추가
3. 키보드 전환: 스페이스바 길게 누르기

## PC 설정

### Windows

1. **설정** → **시간 및 언어** → **언어 및 지역** → **언어 추가**
2. **일본어** 추가
3. 전환: `Win + Space` 또는 `Alt + Shift`

### Mac

1. **시스템 설정** → **키보드** → **입력 소스** → **편집**
2. **일본어 - 로마자** 추가
3. 전환: `Ctrl + Space` 또는 메뉴바 입력 소스 클릭

## 로마자 입력법

일본어 키보드에서 로마자를 치면 자동으로 히라가나로 변환됩니다!

### 기본 입력

| 입력 | 결과 | | 입력 | 결과 |
|------|------|-|------|------|
| a | あ | | ka | か |
| i | い | | ki | き |
| u | う | | ku | く |
| e | え | | ke | け |
| o | お | | ko | こ |

### 주의할 입력

| 입력 | 결과 | 설명 |
|------|------|------|
| shi | し | si도 가능 |
| chi | ち | ti도 가능 |
| tsu | つ | tu도 가능 |
| fu | ふ | hu도 가능 |
| n | ん | 뒤에 자음이 오면 자동 |
| nn | ん | 모음 앞에서는 nn으로 |

### 작은 글자 (촉음/요음)

| 입력 | 결과 | 설명 |
|------|------|------|
| kya | きゃ | k + ya |
| shu | しゅ | sh + u |
| cho | ちょ | ch + o |
| tte | って | 자음 두번 = 촉음 |
| kko | っこ | 자음 두번 = 촉음 |

### 카타카나 변환

히라가나를 입력한 후 **스페이스바**를 누르면 변환 후보가 나와요. 카타카나를 선택하면 됩니다!

| 입력 | 스페이스 → | 선택 |
|------|-----------|------|
| raib u | らいぶ → | ライブ |
| cheki | ちぇき → | チェキ |
| aidoru | あいどる → | アイドル |

## 실전 연습: SNS 댓글 달기

큐티스트리트 SNS에 일본어로 댓글을 달아봅시다!

### 자주 쓰는 댓글

| 한국어 | 입력할 로마자 | 결과 |
|--------|-------------|------|
| 귀여워! | kawaii! | かわいい！ |
| 최고! | saikou! | 最高！ |
| 응원해요 | ouenshitemasu | 応援してます |
| 좋아해요 | sukidesu | 好きです |
| 기대돼요 | tanoshimidesu | 楽しみです |
| 힘내세요 | ganbattekudasai | 頑張ってください |

### 해시태그 입력

| 해시태그 | 입력 |
|----------|------|
| #キューティーストリート | #kyu-thi-sutorito → 변환 |
| #かわいい | #kawaii |
| #推し | #oshi → 변환 |
| #ライブ | #raibu → 변환 |
