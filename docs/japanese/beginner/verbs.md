# 동사 활용 (動詞の活用)

일본어 동사의 기본 활용을 배워봅시다! 큐티스트리트 덕질에 바로 쓸 수 있는 동사로 연습해요.

## 동사의 종류

일본어 동사는 3그룹으로 나뉩니다.

| 그룹 | 특징 | 예시 |
|------|------|------|
| **1그룹** (5단) | ~u로 끝남 | 行く(이쿠/가다), 歌う(우타우/노래하다) |
| **2그룹** (1단) | ~iru/~eru로 끝남 | 見る(미루/보다), 食べる(타베루/먹다) |
| **3그룹** (불규칙) | する, 来る만 | する(스루/하다), 来る(쿠루/오다) |

## ます형 (정중형)

「ます」를 붙이면 정중한 표현이 됩니다. 팬레터나 SNS에서 자주 써요!

### 만드는 법

| 그룹 | 규칙 | 예시 |
|------|------|------|
| 1그룹 | ~u → ~i + ます | 行く → 行き**ます** |
| 2그룹 | ~る 떼고 + ます | 見る → 見**ます** |
| 3그룹 | 외우기 | する → し**ます** / 来る → 来(き)**ます** |

### 덕질 동사 ます형

| 원형 | ます형 | 발음 | 뜻 |
|------|--------|------|-----|
| 行く | 行きます | 이키마스 | 가요 |
| 見る | 見ます | 미마스 | 봐요 |
| 聞く | 聞きます | 키키마스 | 들어요 |
| 歌う | 歌います | 우타이마스 | 노래해요 |
| 踊る | 踊ります | 오도리마스 | 춤춰요 |
| 買う | 買います | 카이마스 | 사요 |
| 撮る | 撮ります | 토리마스 | 찍어요 |
| 書く | 書きます | 카키마스 | 써요 |
| 応援する | 応援します | 오엔시마스 | 응원해요 |
| 推す | 推します | 오시마스 | 최애로 밀어요 |

### ます형 활용표

| 형태 | 만드는 법 | 예 (行く) | 뜻 |
|------|----------|-----------|-----|
| 긍정 현재 | ~ます | 行きます | 가요 |
| 부정 현재 | ~ません | 行きません | 안 가요 |
| 긍정 과거 | ~ました | 行きました | 갔어요 |
| 부정 과거 | ~ませんでした | 行きませんでした | 안 갔어요 |

### 예문

```
ライブに行きます。
(라이브니 이키마스.)
라이브에 가요.

毎日キューティーストリートの曲を聞きます。
(마이니치 큐티스트리토노 쿄쿠오 키키마스.)
매일 큐티스트리트의 곡을 들어요.

遥花ちゃんを応援します！
(하루카짱오 오엔시마스!)
하루카를 응원해요!
```

## て형 (연결형)

「て」형은 일본어에서 가장 많이 쓰이는 활용! ~하고, ~해서, ~해주세요 등에 사용됩니다.

### 만드는 법 (1그룹)

| 원형 끝 | て형 | 예시 |
|---------|------|------|
| ~く | ~いて | 聞く → 聞い**て** |
| ~ぐ | ~いで | 泳ぐ → 泳い**で** |
| ~す | ~して | 推す → 推し**て** |
| ~む/ぶ/ぬ | ~んで | 読む → 読ん**で** |
| ~う/つ/る | ~って | 買う → 買っ**て** |
| 行く (예외) | 行っ**て** | |

### 만드는 법 (2·3그룹)

| 그룹 | 규칙 | 예시 |
|------|------|------|
| 2그룹 | ~る → ~て | 見る → 見**て** |
| する | して | 応援する → 応援し**て** |
| 来る | 来(き)**て** | |

### て형 사용법

| 패턴 | 뜻 | 예문 |
|------|-----|------|
| ~てください | ~해주세요 | 歌ってください (노래해주세요) |
| ~ている | ~하고 있다 | 応援しています (응원하고 있어요) |
| ~てもいい | ~해도 돼요 | 写真を撮ってもいいですか？(사진 찍어도 돼요?) |

### 현장에서 쓰는 て형 예문

```
一緒に歌ってください！
(잇쇼니 우탓테쿠다사이!)
함께 노래해주세요!

チェキを撮ってもいいですか？
(체키오 톳테모 이이데스카?)
체키를 찍어도 될까요?

いつも応援しています。
(이츠모 오엔시테이마스.)
항상 응원하고 있어요.
```
