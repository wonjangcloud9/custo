## 실전 SNS 해석

```
今日のライブやばかった😭✨
キュスト尊すぎて泣いた
遥花ちゃんのMCでジーンときた
ドキドキが止まらない💕
```

→ 해석:
- 오늘 라이브 대박이었어
- 큐스트 너무 존귀해서 울었어
- 하루카의 MC에서 뭉클했어
- 두근두근이 멈추지 않아
