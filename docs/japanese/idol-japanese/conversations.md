# 현장 회화 (現場会話)

큐티스트리트 라이브, 특전회, 물판에서 실제로 쓸 수 있는 회화를 상황별로 배워봅시다!

## 물판 (物販)에서

굿즈를 살 때 쓰는 표현입니다.

| 상황 | 일본어 | 발음 | 뜻 |
|------|--------|------|-----|
| 주문 | これをください | 코레오 쿠다사이 | 이거 주세요 |
| 수량 | 2つお願いします | 후타츠 오네가이시마스 | 2개 부탁합니다 |
| 가격 확인 | いくらですか？ | 이쿠라데스카? | 얼마예요? |
| 결제 | カードで払えますか？ | 카도데 하라에마스카? | 카드로 결제돼요? |
| 감사 | ありがとうございます | 아리가토 고자이마스 | 감사합니다 |

### 물판 롤플레이

```
あなた: すみません、このTシャツをください。
(스미마센, 코노 티셔츠오 쿠다사이.)
실례합니다, 이 티셔츠 주세요.

スタッフ: サイズはどれにしますか？
(사이즈와 도레니시마스카?)
사이즈는 어떤 걸로 하시겠어요?

あなた: Mサイズでお願いします。
(엠사이즈데 오네가이시마스.)
M사이즈로 부탁합니다.

スタッフ: 2000円になります。
(니센엔니 나리마스.)
2000엔입니다.
```

## 특전회 (特典会)에서

체키나 사인회에서 멤버와 대화할 때!

| 상황 | 일본어 | 발음 | 뜻 |
|------|--------|------|-----|
| 인사 | はじめまして！ | 하지메마시테! | 처음 뵙겠습니다! |
| 자기소개 | 韓国から来ました | 한코쿠카라 키마시타 | 한국에서 왔어요 |
| 칭찬 | 今日もかわいいです！ | 쿄모 카와이이데스! | 오늘도 귀여워요! |
| 공연 감상 | ライブ最高でした！ | 라이브 사이코데시타! | 라이브 최고였어요! |
| 응원 | ずっと応援します！ | 즛토 오엔시마스! | 계속 응원할게요! |
| 사진 요청 | ピースしてください！ | 피스시테쿠다사이! | 피스 해주세요! |
| 작별 | また会いに来ます！ | 마타 아이니 키마스! | 또 만나러 올게요! |

### 특전회 롤플레이

```
あなた: はじめまして！韓国から来ました。
(하지메마시테! 한코쿠카라 키마시타.)

メンバー: えー！韓国！ありがとう！
(에~! 한코쿠! 아리가토!)

あなた: 遥花ちゃんのダンスが大好きです！
(하루카짱노 단스가 다이스키데스!)
하루카의 춤을 정말 좋아해요!

あなた: ライブ最高でした！
(라이브 사이코데시타!)
라이브 최고였어요!

あなた: ピースしてください！
(피스시테쿠다사이!)
피스 해주세요!

あなた: また会いに来ます！ずっと応援します！
(마타 아이니 키마스! 즛토 오엔시마스!)
또 만나러 올게요! 계속 응원할게요!
```

## 라이브 회장에서

입장부터 퇴장까지 알아두면 좋은 표현!

| 상황 | 일본어 | 발음 | 뜻 |
|------|--------|------|-----|
| 입장 | 整理番号○番です | 세이리방고 ○방데스 | 정리번호 ○번이에요 |
| 자리 확인 | この席ですか？ | 코노 세키데스카? | 이 자리인가요? |
| 화장실 | トイレはどこですか？ | 토이레와 도코데스카? | 화장실 어디예요? |
| 감상 | 楽しかった！ | 타노시캇타! | 즐거웠어! |
| 감동 | 感動しました | 칸도시마시타 | 감동했어요 |
| 앙코르 | アンコール！ | 앙코루! | 앙코르! |

## 일반 이동/식사

일본 원정 때 필수 표현!

| 상황 | 일본어 | 발음 | 뜻 |
|------|--------|------|-----|
| 길 묻기 | ○○はどこですか？ | ○○와 도코데스카? | ○○는 어디예요? |
| 주문 | 注文お願いします | 추몬 오네가이시마스 | 주문 부탁합니다 |
| 추천 | おすすめは何ですか？ | 오스스메와 난데스카? | 추천은 뭐예요? |
| 계산 | お会計お願いします | 오카이케이 오네가이시마스 | 계산 부탁합니다 |
