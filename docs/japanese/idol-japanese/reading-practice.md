# SNS 읽기 연습 (読む練習)

큐티스트리트의 SNS 포스트를 읽어보면서 일본어 실력을 테스트해봅시다!

## 연습 1: 공연 공지

다음 포스트를 읽고 질문에 답해보세요.

```
📢 お知らせ

12月15日(日)
キューティーストリート ワンマンライブ
「Happy Sekai! Release Party」

会場: 渋谷WWW
開場: 17:00 / 開演: 18:00
チケット: 3,500円

皆さん、会いに来てね！💕
```

|------|------|-----|

| 開場 | 카이조 | 개장 (문 오픈) |
