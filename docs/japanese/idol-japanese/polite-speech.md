# 정중한 표현 (丁寧な表現)

특전회에서 멤버에게, 팬레터에서, 스태프에게 정중하게 말하는 법을 배워봅시다!

## 정중도 레벨

일본어는 상황에 따라 정중도가 달라요.

| 레벨 | 예시 | 사용 상황 |
|------|------|----------|
| 반말 (タメ語) | 行く | 친한 친구끼리 |
| 정중체 (丁寧語) | 行きます | 일반적인 대화 |
| 존경어 (尊敬語) | いらっしゃいます | 상대를 높일 때 |
| 겸양어 (謙譲語) | 参ります | 자신을 낮출 때 |

> 💡 팬으로서 멤버에게는 **정중체**(ます형)를 쓰면 충분해요!

## 특전회에서 쓰는 정중 표현

### 기본 정중 표현

| 일반 | 정중 | 발음 | 뜻 |
|------|------|------|-----|
| ありがとう | ありがとうございます | 아리가토 고자이마스 | 감사합니다 |
| すみません | 申し訳ありません | 모시와케아리마센 | 죄송합니다 |
| いい？ | よろしいですか？ | 요로시이데스카? | 괜찮으시겠어요? |
| 見て | ご覧ください | 고란쿠다사이 | 봐주세요 |
| 待って | お待ちください | 오마치쿠다사이 | 기다려주세요 |

### お/ご + 명사 (정중 접두어)

| 일반 | 정중 | 뜻 |
|------|------|-----|
| 名前 | **お**名前 | 이름 (정중) |
| 写真 | **お**写真 | 사진 (정중) |
| 元気 | **お**元気 | 건강 (정중) |
| 活動 | **ご**活動 | 활동 (정중) |

> 💡 일본어 고유어에는 「お」, 한자어에는 「ご」를 붙여요!

## 특전회 정중 회화

### 처음 만날 때

```
はじめまして。
いつもSNSで応援させていただいています。
(하지메마시테. 이츠모 에스엔에스데 오엔사세테이타다이테이마스.)
처음 뵙겠습니다. 항상 SNS에서 응원하고 있습니다.
```

### 칭찬할 때

```
今日のライブ、とても素敵でした。
(쿄노 라이브, 토테모 스테키데시타.)
오늘 라이브, 정말 멋졌어요.

ダンスがとてもお上手ですね。
(단스가 토테모 오조즈데스네.)
춤을 정말 잘추시네요.
```

### 부탁할 때

```
お写真を撮っていただけますか？
(오샤신오 톳테이타다케마스카?)
사진을 찍어주실 수 있을까요?

ピースをしていただけますか？
(피스오 시테이타다케마스카?)
피스를 해주실 수 있을까요?
```

### 헤어질 때

```
お体に気をつけてくださいね。
(오카라다니 키오츠케테쿠다사이네.)
몸 건강 조심하세요.

これからも応援させていただきます。
(코레카라모 오엔사세테이타다키마스.)
앞으로도 응원하겠습니다.
```

## ~ていただけますか vs ~てください

| 표현 | 정중도 | 예문 | 뜻 |
|------|--------|------|-----|
| ~て | ★ | 撮って | 찍어 |
| ~てください | ★★ | 撮ってください | 찍어주세요 |
| ~ていただけますか | ★★★ | 撮っていただけますか | 찍어주실 수 있을까요? |

> 💡 특전회에서는 「~てください」면 충분히 정중해요!

## 팬레터 정중 표현

| 일반 | 팬레터용 | 뜻 |
|------|----------|-----|
| 好きです | 大好きです | 정말 좋아합니다 |
| 頑張って | 頑張ってください | 힘내세요 |
| 応援してる | 応援しています | 응원하고 있습니다 |
| また会いたい | またお会いしたいです | 다시 뵙고 싶습니다 |
| 元気？ | お元気ですか？ | 건강하신가요? |
